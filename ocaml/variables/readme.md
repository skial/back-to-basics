# Variables

## Default characteristics

## Creation
- `let <name>`

## Assignment
- `let <name> = <value>`

## Re-assignment
- `<name> = `<new value>` is not allowed in OCaml, redefine the variable instead.

## Re-define
- ```ocaml
  let v = 10
  let v = "foo bar"
  ```
- Can be redefined with a new type.
